Welcome to My Math Page

This is a demo site showing how to use LaTeX equations with MathJax on GitHub Pages.

Bullet Points

  • Inline math example: \( a^2 + b^2 = c^2 \)
  • You can write Greek letters like \( \alpha, \beta, \gamma \)
  • Summations: \( \sum_{i=1}^n i = \frac{n(n+1)}{2} \)

Block Equations

\[ E = mc^2 \]
\[ \int_{-\infty}^{\infty} e^{-x^2}\, dx = \sqrt{\pi} \]

Some Explanatory Text

Equations can appear inline like this one: \( f(x) = \sin(x) \), or as a centered block like the ones above. You can also mix text, lists, and equations freely.
